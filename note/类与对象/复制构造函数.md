## 定义
1.一个特殊的构造函数;  
2.形参为本类对象的引用;  
3.作用是使用一个已经存在的对象去初始化一个新对象;
## 隐含复制构造函数
1.类似于构造函数,在没有定义类的复制构造函数的时候,编译器就会自动生成一个隐含的复制构造函数;  
2.*功能*  
那初始值对象的每个数据成员的值都复制到新建立的对象中;
```
Clock s = Clock(myclock);//隐含的复制构造函数
```
## 自定义
1.一般方法:
```
class 类名
{
    public :
        类名(形参表);
        类名(类名&对象名);
        ...
};
类名::类名(类名&对象名)
{
    函数体;
} 
```
2.具体例子:
```
class point{
    public :
        Point(int x1,iny y1=0)
        {
            x=x1;
            y=y1;
        }
        Point(Point &p);
        ...
    private :
        int x,y;
}
Point::Point(Point &p)
{
    x=p.x;
    y=p.y;
}
```
##  使用情况
1.用一个对象去初始化该类另一个对象
```
Point b = Point(a);
```
2. 按值传递,实参为对象;
```
void f(Point a);//实参转换为形参时会调用
int main()
{
    Point a(1,2);
    f(a);
}
```
3.函数返回值为对象;
```
Point g()
{
    Point a(1,2);
    return a;//返回对象时,临时类变量会解除,调用构造函数返回
}
```
