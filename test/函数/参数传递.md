## 1.按值传递
### **特点**
实参值初始化形参,形参的改变不会实参;
### **形式**
```
void function(type a,type b)
```
## 2.引用传递
### **特点**
引用相当变量的一个别名,访问同一个地址;
### **形式**
```
int r=0;
int &ri=r;//ri为r的引用
```
```
void function(type &a)
```
## 3.指针传递
### **特点**
实参为地址,形参为指向实参地址的指针;  
其实,本质还是按值传递,只不过传递的是地址;
### **形式**
```
function(&a);
void function(*a);
```
